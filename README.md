# Proyecto PickyUp (Web & Móvil) - Proyecto02-DS

## Inicio Rápido

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/NathaliaAlf/Proyecto02-DS.git
   cd Proyecto02-DS/Dev-Web-Mob
   ```

2. **Trabajar en la rama de development**
   ```bash
   git checkout development
   ```

3. **Instalar dependencias:**
   ```bash
   npm install
   ```

4. **Ejecutar el proyecto:**
    - **Para Web:** `npx expo start --web`
    - **Para Móvil (Expo Go):** `npx expo start`
